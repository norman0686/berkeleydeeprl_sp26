## Git Commands

Forked form https://github.com/berkeleydeeprlcourse/homework_spring2026.

Created by

```bash
$ git clone https://github.com/norman0686/berkeleydeeprl_sp26 .
```

```bash
$ git remote add upstream https://github.com/berkeleydeeprlcourse/homework_spring2026
```

To Verify remotes

```bash
$ git remote -v
```

To pull future updates from the original repo:

```bash
$ git pull upstream main
```
